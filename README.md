# Detox YT 🧘‍♂️

**Detox YT** is a sleek, modern Chrome extension designed to transform YouTube from a distraction machine into a highly focused learning tool. By setting a "Focus Topic," this extension completely filters your YouTube feed, hiding irrelevant videos, Shorts, and sidebars, so you only see content related to your current goals.

## ✨ Features

* **Laser Focus Mode:** Enter a topic (e.g., "Next.js Tutorials" or "React Native") and instantly filter your YouTube feed to only show videos matching your focus.
* **Distraction Free:** Automatically hides YouTube Shorts, the main/mini sidebars, comment sections, and top-of-feed ads.
* **Modern UI:** A sleek, dark-themed popup interface with a native iOS-style toggle switch and Tailwind-powered styling.
* **Quick Toggle:** Easily pause Detox mode with a single click if you want to browse YouTube normally, complete with visual UI dimming when disabled.
* **Smart History:** Keeps track of your watched videos and caches fetched topics to ensure a smooth, fast browsing experience.

## 🛠️ Tech Stack

This extension is built using standard modern web technologies:

* **Frontend Framework:** React 18
* **Styling:** Tailwind CSS
* **Language:** TypeScript
* **Bundler:** Webpack
* **APIs:** Chrome Extension Manifest V3 (Storage, Runtime, Tabs APIs)

## 🚀 Installation (Load Unpacked)

Since this extension is currently unlisted on the Chrome Web Store, you can easily install it locally on your machine for free:

1. **Clone the repository:**
  https://github.com/ankitpwr/DetoxYT.git
2. **Install dependency**
   npm install
3. **Build extension**
   npm run build
4. **developemnt**
   npm run start
